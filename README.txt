# 实时工资计算器 - 测试运行指南

## 一、启动服务器

如果服务器尚未启动，请按照以下步骤启动：

1. 打开命令提示符或终端
2. 导航到项目目录：`d:\try\make money`
3. 运行命令：`npx --yes http-server -p 8000 -c-1`
4. 服务器将在 http://localhost:8000 上启动

## 二、访问应用程序

1. 打开您的浏览器
2. 在地址栏输入：http://localhost:8000
3. 您将看到实时工资计算器的界面

## 三、测试数据设置

为了快速看到效果，建议使用以下测试数据：

- **月薪**：100000 元（高薪资可以更快看到金币掉落效果）
- **每月工作天数**：22 天
- **上班时间**：09:00
- **下班时间**：18:00

## 四、功能测试

### 1. 基本计算功能
- 输入上述测试数据
- 点击「开始计算」按钮
- 观察实时工资数字是否每秒更新

### 2. 金币掉落功能
- 当工资达到 10 元、20 元、30 元等 10 的整数倍时
- 会从屏幕顶部掉落金色的金币动画

### 3. 字幕显示功能
- 当金币掉落时，屏幕中央会显示金色字幕：
  "你已经挣了XX元！"
- 字幕会有淡入淡出的动画效果

### 4. 音效播放功能
- 当金币掉落时，会播放连续 3 次「咣咣咣」的音效
- 首次使用时需要点击页面任意位置激活音效（浏览器安全要求）

### 5. 重置功能
- 点击「重置」按钮
- 所有数据将清空，计算停止

## 五、调试模式

如果需要查看详细的运行日志：

1. 在浏览器中按 F12 打开开发者工具
2. 切换到「控制台」选项卡
3. 可以看到各种功能的执行日志，如：
   - 「开始计算按钮被点击」
   - 「显示字幕: 10」
   - 「播放音效」

## 六、常见问题解决

1. **没有音效？**
   - 首次使用需要点击页面任意位置激活音频
   - 检查浏览器音量设置

2. **没有字幕显示？**
   - 确保工资达到 10 元的整数倍
   - 检查浏览器控制台是否有错误信息

3. **页面没有反应？**
   - 确保服务器正在运行
   - 尝试刷新页面（按 Ctrl+F5 强制刷新）
   - 检查浏览器控制台的错误信息

## 七、停止服务器

1. 回到启动服务器的命令提示符或终端
2. 按 Ctrl+C 停止服务器
